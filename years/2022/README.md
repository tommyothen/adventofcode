# Advent of Code 2022

This directory contains my solutions to the 2022 [Advent of Code](https://adventofcode.com/) puzzles.

## Language
For 2022, I will be using [Rust](https://www.rust-lang.org/).

I've been wanting to learn Rust for quite some time now so I figured this would be a good opportunity to do so.

## Building and running
To build and run the solutions, you will need to have [Rust](https://www.rust-lang.org/) installed. We also use [cargo-single](https://crates.io/crates/cargo-single) to run the solutions.

To build and run the solutions, you can use the following commands:

```bash
# Run the solution for day 1
cargo single run day1/solution.rs
```

Running the solutions will create a `solution` subdirectory within the respective day directory. This directory will contain the compiled binary.
