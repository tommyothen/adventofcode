# 🎄 Advent of Code

This repository contains my solutions to the [Advent of Code](https://adventofcode.com/) puzzles.

Every year, I will try to solve the puzzles in a different language.
This will vary from year to year, but I will try to stick to languages that I am not familiar with.

The goal is to learn something new every year, or at least to have fun trying :)

All languages before 2022 are written in languages of a level that I would consider to be beginner or intermediate at the time of release. They are also chosen in order from when I first learned them.

## Languages

* [2022](years/2022/README.md) - [Rust](https://www.rust-lang.org/)
* [2021](years/2021/README.md) - [C++](https://isocpp.org/)
* [2020](years/2020/README.md) - [Haskell](https://www.haskell.org/)
* [2019](years/2019/README.md) - [Lua](https://www.lua.org/)
* [2018](years/2018/README.md) - [TypeScript](https://www.typescriptlang.org/)
* [2017](years/2017/README.md) - [Java](https://www.java.com/en/)
* [2016](years/2016/README.md) - [JavaScript](https://www.javascript.com/)
* [2015](years/2015/README.md) - [Python](https://www.python.org/)

## Structure
The repository is structured as follows:

* `years` - Contains a folder for each year
* `years/<year>` - Contains a folder for each day
* `years/<year>/<day>` - Contains the solution for the day
